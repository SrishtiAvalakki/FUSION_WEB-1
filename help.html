Frequently asked questions

        You can create groups using two options: 
            1.Private groups
            2.Public group
        You can choose either of the options and select the users from the list of the users displayed and then click on submit.
    

To search for various users that exist, on the profile page of roomies website, click  on the search users tab on the navigation bar. This will redirect you to the search engine, where you can type the name of the users and click on the name which will take you to that user's respective profile page.
    

        To view the other user's profile, click on the user under the search engine bar and the ratings of the user based on 2 criterias will appear:
            a.The number of groups that the user is a part of.
            b.The number of posts the respective user has posted.
    

To upload the images and files onto the posting area ,select the respective file or the image by clicking on the "Choose File"  button,  and click on the "POST" button. The respective image and file would have been uploaded onto the posting area.
